# 埋点及用户交互PRD
### 埋点采集对象
- 用户行为数据
- 部分页面性能数据
- 用户在线状态
- app错误信息
### 埋点需求
##### 用户行为数据
- 每个页面的进入事件（用于分析页面的使用频率）
- 重点按钮的点击事件（用户分析某些功能的使用频率）
##### 部分页面性能数据
- 课表页渲染耗时（用于分析课表页加载速度，课表功能是核心功能）
- 部分列表页渲染耗时（用于分析长列表性能）
##### 用户在线状态
- 在线状态（用于查看实时用户在线情况，挂后台也算在线，只要没有彻底退出应用）
- 用户日活（用于分析app日活情况）
##### app错误信息
- 捕获app崩溃信息（用于分析应用崩溃原因）
### 用户交互
##### 用户建议和反馈
- 想要的功能建议（如新增功能、优化功能等）
- 反馈问题（如应用崩溃、性能问题、bug等）
