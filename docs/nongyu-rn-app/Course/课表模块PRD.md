# 课表模块PRD
### 基础课表周视图
以周为单位，展示当周课程情况，每周展示7天，每天有五个大课区间
共有35个大课区间，对应最多可插入35个课程卡片
课程卡片的颜色各异，有区分度，方便用户识别
用户也可以配置课程卡片统一颜色，无需区分不同课程的颜色
对于连堂课程，卡片应该连接，类似于合并单元格的效果
每个课程卡片上只显示课程名称、教师姓名、教室位置
当点击课程卡片时，打开一个从下往上弹出的大半屏弹窗，展示课程详情，弹窗内容见下方课程详情弹窗
课程卡片的hover和点击应该有一定动效提高用户体验，例如卡片颜色变化、阴影变化等
课表打开时，默认展示当前周的课程
用户可以通过左右滑动切换到其他周的课程
如果当前展示的是非本周课程，提供一个悬浮按钮，点击后切换到本周课程
点击悬浮按钮后，课表会切换到本周课程的周视图
周视图顶部展示周几和当前是第几周，还需要标明周几是几月几号，当日的日期应该有强调效果
用户可以配置是否高亮展示当日课程所在的那一列
周视图背景可以设置照片
用户可以且需要自行设置正确的开学日期，课表会根据开学日期展示课程
### 自定义课程
用户可以自定义课程，包括课程名称、教师姓名、教室位置、时间、地点等信息，但只能添加到没有课程的大课区间中，如果当前大课区间有课程，用户不能添加课程到该大课区间中，需要给用户提示错误信息
用户可以删除已添加的课程包括初始时系统自动抓取的用户本学期课程，删除按钮位于课程详情弹窗的底部，用户也可以编辑已添加的课程信息
### 课程详情弹窗
- 展示课程基本信息，包括课程名称、教师姓名、教室位置、时间、地点等信息
- 展示课程考勤记录、待办事项、备注信息、上课提醒等扩展功能
### 自定义日程
除课程以外，在每个大课区间中，用户可以添加自定义的日程，例如：
- 日程名字
- 日程内容
- 日程时间
- 日程地点
- 其他信息
用户可以删除和编辑已添加的日程信息
同样的，用户不能添加到有课程的大课区间中，需要给用户提示错误信息
日程被添加后，会和课程卡片一样显示在周视图中，点击后的详情弹窗内容见下方日程详情弹窗
### 日程详情弹窗
- 展示日程基本信息，包括日程名字、日程内容、日程时间、日程地点、其他信息等
### 扩展功能
- 课程考勤记录（用户可以为每个课程记录该课程的考勤情况，包括签到、迟到、缺勤、请假）
- 课程待办事项（用户可以为每个课程添加待办事项，例如：完成项目、提交作业等）
- 课程备注信息（用户可以为每个课程添加备注信息，例如：课程内容、教师备注等）
- 课程上课提醒（用户可以为该课程添加上课提醒，即在课程开始前通知用户，帮助用户准备上课，若无好的技术方案则暂不实现该功能）

### 课表海报
用户可以导出当前周的课表海报，海报上展示当前周的课程情况，包括课程名称、教师姓名、教室位置、时间、地点等信息

### 分享课表
用户之间可以便捷分享各自没有课的时间段便于安排用户之间的其他活动
需求背景：
用户之间想要远程分享各自课表，但一张张的截图分享方式比较麻烦，需要用户手动操作，而系统可以提供一个便捷的分享方式，例如：
生成课表二维码，用户可以扫描二维码分享课表
