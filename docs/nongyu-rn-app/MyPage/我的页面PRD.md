# 我的页面PRD
### 个人基本信息展示和编辑
- 展示用户个人基本信息，参见[用户登录登出PRD](../User/登录登出PRD.md)
- 编辑用户个人基本信息，暂时只可编辑QQ，参见[用户登录登出PRD](../User/登录登出PRD.md)
### 登录登出入口
“我的”页面是用户登录和登出的入口页面，用户可以在“我的”页面登录和登出。
- 登录入口：用户点击“登录”按钮，跳转到登录页面，参见[登录登出PRD](../User/登录登出PRD.md)
- 登出入口：用户点击“登出”按钮，跳转到登录页面，参见[登录登出PRD](../User/登录登出PRD.md)
### 设置页入口
“我的”页面仅展示设置入口，用户点击设置入口，跳转到设置页面，参见[设置页面PRD](./设置页面PRD.md)
### 我的评论、回复和发帖
“我的”页面仅展示入口，用户点击入口，跳转到对应页面，参见[用户评论回复发帖PRD](./用户评论回复发帖PRD.md)
