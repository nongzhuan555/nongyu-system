# 农屿系统需求大纲
## 系统架构
以nongyu-rn-app为用户端主站，nongyu-node-server为单体后端，nongyu-web-admin为后台管理端，nongyu-web-site为农屿品牌官网
详情：
- nongyu-rn-app（目标用户为四川农业大学在读本科生，目标覆盖率为50%或以上）
- nongyu-node-server（农屿后端，提供API接口）
- nongyu-web-admin（目标用户为农屿系统管理员，包括开发团队本身和部分农屿用户）
- nongyu-web-site（目标用户为农屿系统用户，用于展示农屿app信息和体验部分产品功能，且提供农屿的下载链接和团队联系方式）
