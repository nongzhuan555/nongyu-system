## 各项目技术栈

### 原则
- 除特殊情况外，项目中用到的同一个依赖统一使用同一个版本，受pnpm-workspace.yaml中的catalog配置管控
- 除特殊情况外，整个项目都基于同一个版本的Node.js进行开发，当前版本为22.x

### Node.js 版本

- 22.x

### 全局依赖（放在根目录的 package.json 中，根据最佳实践，只放开发时依赖）
开发时依赖：
- 语言（TypeScript，用于全局中所有用到ts的项目）
- TS打包器（tsup，用于packages下的项目）
- 代码格式化（Oxfmt）
- 代码质量工具（Oxlint）
- 构建工具（Vite）

### nongyu-web-admin

- 网络（axios）
- 骨架（react 18）
- UI（ant design）
- 图表（echarts）
- 样式（Tailwind CSS）
- 路由（react router）
- 语言（TS）
- 状态管理（Zustand）
- 构建工具（Vite8）

### nongyu-node-server

- 数据库（MySQL）
- 框架（Express）
- 语言（TypeScript）
- 数据库驱动（mysql2）
- 进程管理（pm2）
- 日志管理（winston）


### nongyu-rn-app

- 框架（React native）
- 语言（TypeScript）
- 构建工具（Expo）
