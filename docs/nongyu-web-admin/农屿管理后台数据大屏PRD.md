# 农屿管理后台数据大屏PRD
### 需求背景
通过可视化图表的形式展示农屿管理员端的数据，帮助管理员清晰直观的了解系统运行状态和用户行为
### 大屏组成部分概览
- 总用户数
- 今日日活用户数
- 在线用户数
- 今日新增用户数
- 用户增长趋势图
- 用户分布数据
- 用户行为数据
- App运行相关数据
### 各组成部分具体要求
##### 总用户数
展示农屿App的总用户数，包括总用户数以及总管理员数，总管理数应该是用户数的子集
##### 今日日活用户数
展示农屿App的今日日活用户数，即今日内打开过农屿App的用户数
##### 当前在线用户数
展示农屿App的当前在线用户数，每次进入app后会把用户状态设为在线，退出app后会把用户状态设为离线，统计当前在线的用户数
##### 今日新增用户数
展示农屿App的今日新增用户数，即今日内注册的用户数，即用户表中create_time在今日内的用户数
##### 用户增长趋势图
展示农屿App的用户增长趋势图，即可选最近7天、30天、90天、半年、一年的用户增长趋势，默认展示最近7天的用户增长趋势
##### 用户分布数据
展示农屿App的用户分布数据，包括：
- 用户性别分布（看男女比例）
- 用户校区分布
- 用户学院分布
- 用户年级分布
- 用户设备分布（品牌分布，苹果就是苹果，安卓细分品牌，如小米、oppo、vivo、荣耀等）
- 用户使用时间分布（看用户在app上的使用时间分布）
##### 用户行为数据
展示农屿App的用户行为数据，包括：
- 各采集页面的使用次数（看哪些页面或功能被使用的最多）
- 用户设置项分布（看用户设置项的分布，如是否首屏打开课表、主题、是否在app内部打开网页等，看用户的使用习惯）
##### App运行相关数据
- 关键性能采集数据展示（课表渲染时间、长列表加载性能）
- 错误捕获展示
