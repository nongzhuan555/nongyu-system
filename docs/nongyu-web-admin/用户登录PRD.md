# 用户登录PRD
### 登录页面
- 展示登录页面，包括教务网学号、农屿密码、登录按钮
- 用户输入教务网学号和农屿密码后，点击登录按钮，系统会验证学号和密码是否正确
- 若用户名和密码正确，系统会跳转到主页面
- 若用户名和密码错误，系统会提示用户输入错误
### 登录类型
- 外部浏览器登录（默认）
- 内部App登录
后端需接受登录类型参数，仅作记录，不区分
### 双重登录
当管理员端以移动端形式嵌入在App内部时，自动填入学号和密码（密码来自于本地设置并存储的农屿管理员体系密码，区别于教务网密码，教务网密码用于登录农屿App，而农屿管理员体系密码用于登录农屿管理员端页面）
农屿App通过本地的学号和管理员密码与Webview嵌入的管理员端页面通信自动填入学号和密码，可以便捷登录无需手动输入，登录类型自动填为内部App登录
### 多点登录
由于农屿管理员端没有本地存储，无需考虑不同设备的本地存储和远程数据的矛盾，用户可以同时在多个设备上登录农屿管理员端页面，不卡号，即不会使其他设备登录失效
